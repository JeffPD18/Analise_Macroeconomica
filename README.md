# Dashboard Macroeconômico do Brasil  
Macroeconomic Dashboard of Brazil

---

## 🇧🇷 Português

Este projeto apresenta um dashboard desenvolvido em **Power BI** para análise de indicadores macroeconômicos brasileiros ao longo dos últimos anos.

O objetivo é explorar a dinâmica entre **inflação, taxa de juros, desemprego e câmbio**, além de analisar relações econômicas como a **Curva de Phillips** e os efeitos da política monetária.

### Indicadores analisados

- Inflação (IPCA)
- Taxa de juros
- Desemprego
- Taxa de câmbio (USD/BRL)
- Juros reais
- Curva de Phillips

### Ferramentas utilizadas

- Power BI
- DAX
- Modelagem de dados
- Visualização de dados

### Fonte dos dados

- IBGE
- Banco Central do Brasil

## 🇺🇸 English

This project presents a **Power BI dashboard** developed to analyze key Brazilian macroeconomic indicators over the past years.

The objective is to explore the dynamics between **inflation, interest rates, unemployment, and exchange rates**, as well as economic relationships such as the **Phillips Curve** and monetary policy effects.

### Indicators analyzed

- Inflation (IPCA)
- Interest rates
- Unemployment
- Exchange rate (USD/BRL)
- Real interest rates
- Phillips Curve

### Tools used

- Power BI
- DAX
- Data modeling
- Data visualization

### Data sources

- Brazilian Institute of Geography and Statistics (IBGE)
- Central Bank of Brazil
